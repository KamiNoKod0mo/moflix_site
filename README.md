# Moflix
Uma aplicação web criada com React.

## Tecnologias
- React 18
- Mysql
- Node

## Requisitos
- Node.js v16+ e npm ou Yarn.

## Como executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/KamiNoKod0mo/moflix_site
   cd moflix_site
   npm install
   npm start
   ```
